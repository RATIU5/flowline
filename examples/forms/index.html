@flowline/forms-vanilla Demo

This example showcases the key features of the @flowline/forms-vanilla library:

  • Typed Input Values: Age field uses data-type="number" for automatic type conversion
  • Custom Error Rendering: Errors display with custom styling and icons
  • Effect-based Architecture: Proper resource management and cleanup
  • Reactive State: Real-time form state display below
  • Validation: Comprehensive field validation with custom messages
