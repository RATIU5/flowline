# Kysely & PG

**Date:** 2026-03-26 | **Phase:** 2

## Context

Need a database and query builder. I want a query builder for type-safe queries and improved developer experience.

## Options

| Option        | Why Consider                    | Why Not                      |
| ------------- | ------------------------------- | ---------------------------- |
| A. Kysely     | Basic query builder             | Need full power of ORM       |
| B. Drizzle    | Full ORM                        | Don't need full ORM          |
| C. PostgreSQL | Flexible & scalable, relational | Need to store more objects   |
| D. MongoDB    | Flexible & scalable, NoSQL      | Need to store more relations |

## Chose: A & C (Kysely & PostgreSQL)

**Why:** PostgreSQL is more relational and I prefer more relational oriented databases. Kysely also supports PostgreSQL out of the box and provides a lightweight type-safe query builder for it.

**Tradeoff accepted:** I am giving up a full-featured ORM like Drizzle for a more lightweight solution. I am also giving up on the flexibility of a NoSQL database like MongoDB in preference of relational data.
